# MelbCSS Modules Hello World

A simple [CSS Modules] example, using [React] with [Webpack]'s [css-loader] in [module mode].

Extracted from a live coding demo at [MelbCSS].

## Run the example

```bash
$ npm install
$ npm start & open http://localhost:8080
```
